# Security Policy

Found something that looks like a security issue in ARN-Lifebar? Thanks for taking the time to flag it — please report it privately so it can be fixed before anything is public.

## How to report

Open a private report via GitHub's Security Advisory form:

→ **https://github.com/jzbakh/antigravity-arn-lifebar/security/advisories/new**

Please don't open a public issue or discussion for security topics.

## What's mine to fix vs. what's upstream

- Anything wrong with **ARN-Lifebar itself** — its code, how it talks to the local Antigravity language server, the data it reads — that's mine and I'll look into it.
- Bugs in **Antigravity IDE** itself, or in its `language_server`, belong upstream with the Antigravity team. If you're not sure where it should go, send it my way and I'll redirect.

## Response time

I'm a solo maintainer. You'll usually hear back within a few business days.
